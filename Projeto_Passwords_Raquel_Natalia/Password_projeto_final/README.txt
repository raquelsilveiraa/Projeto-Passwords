Projeto passaword 

Instruções para funcionamento do projeto:

Inicialmente deve ser colocado o arquivo "password.csv" na pasta "dados_entrada".

Em seguida, deve ser iniciado pela funcao "Classifica_password.java", que vai classificar as senhas. 
Em seguida, deve ser executado o  "transformacoes.java", que ira filtras as senhas para "muito boa e boa" e vai transformar a formatacao da data de "YYYY-MM-DD" para "DD/MM/YYYY".
Em seguida, deve ser executado a "Main_length", que vai ordenar o array a partir do tamanho da senha, para os casos "Melhor, Médio e Pior", com diferentes algoritmos de ordenação (Bubblesort, InsertionSort, HeapSort, MergeSort, Quicksort, Quicksort mediana de três, SelectionSort e CountingSort).
Em seguida, deve ser executado a "Main_data", que vai ordenar o array a partir do data, para os casos "Melhor, Médio e Pior", com diferentes algoritmos de ordenação (Bubblesort, InsertionSort, HeapSort, MergeSort, Quicksort, Quicksort mediana de três, SelectionSort e CountingSort).
Por fim, deve ser executado a "Main_month", que vai ordenar o array a partir do mês, para os casos "Melhor, Médio e Pior", com diferentes algoritmos de ordenação (Bubblesort, InsertionSort, HeapSort, MergeSort, Quicksort, Quicksort mediana de três, SelectionSort e CountingSort).

Os resultados se encontram na pasta "dados_saida".

O programa "leitor_entrada", lê a entrada. Já "Dados_password", extrai os dados da entrada, para serem utilizados.
